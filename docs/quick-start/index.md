# 快速开始

这里帮助你先选对使用方式，再进入对应的最短流程。详细参数、完整安装说明和故障排查仍放在各自的 Notebook、整合包、下载器、Launcher 和安装器文档中。

## 先选使用方式

| 你想做什么 | 推荐路径 | 适合人群 |
| --- | --- | --- |
| 不想配置本地环境，直接在云端运行 | [云端运行](./cloud.md) | Colab / Kaggle 用户，临时体验或训练 |
| Windows 本地下载即用 | [整合包快速开始](./portable.md) | 想少配置、快速解压使用 |
| Windows 图形界面安装和管理 | [Launcher 快速开始](./launcher.md) | 想用一个入口安装、启动、更新和管理 |
| Windows / Linux / macOS 本地安装 | [本地安装](./local-install.md) | 需要完整本地部署和长期维护 |
| 训练模型或运行训练脚本 | [云端运行](./cloud.md) 或 [本地安装](./local-install.md) | SD Trainer / sd-scripts 用户 |
| AMD 显卡在 Windows 上运行 | [整合包快速开始](./portable.md) 后阅读 [AMD ZLUDA 使用](../portable/zluda.md) | RX / 780M / 680M 等 AMD 用户 |

## 推荐路线

新手优先选择这三条路线：

1. 想最快看到 WebUI：阅读 [云端运行](./cloud.md)，使用 Colab Notebook，默认参数可直接运行。
2. 想在 Windows 本地免安装使用：阅读 [整合包快速开始](./portable.md)，下载 Nightly 整合包并解压。
3. 想长期安装和维护：阅读 [Launcher 快速开始](./launcher.md)，用 Launcher 安装或接管已有目录。

如果已经成功启动 WebUI，继续阅读 [启动后的下一步](./after-start.md)，了解模型下载、管理脚本和 SD Note 使用教程入口。
