# 整合包与实用指南

本板块整理本地 Windows 用户常用的整合包、AMD ZLUDA 和常见问题文档。整合包适合希望下载后直接解压使用的用户；ZLUDA 教程适合需要在 Windows 上使用 AMD 显卡运行绘图工具的用户。

## 推荐阅读顺序

1. 需要本地免安装使用 WebUI：先阅读 [整合包下载与使用](./portable.md)。
2. 使用 AMD 显卡并需要在 Windows 上运行：阅读 [AMD ZLUDA 使用](./zluda.md)。
3. 启动脚本或绘世启动器遇到问题：阅读 [常见问题](./qa.md)。

## 文档说明

- [整合包下载与使用](./portable.md)：包含整合包下载器、各产品整合包入口和常用管理脚本。
- [AMD ZLUDA 使用](./zluda.md)：包含 HIP SDK、PyTorch 重装、780M / 680M 额外配置、绘世启动器启动和跑图测试。
- [常见问题](./qa.md)：记录本地运行时常见问题和处理方式。
