# 整合包与实用指南

本板块整理本地 Windows 用户常用的整合包、AMD ZLUDA 和常见问题文档。整合包适合希望下载后直接解压使用的用户；它基于 Installer 构建，下载并解压后默认通过 Installer 生成的脚本启动和维护，也可以用 Launcher 指向解压目录来接管管理。ZLUDA 教程适合需要在 Windows 上使用 AMD 显卡运行绘图工具的用户。

## 推荐阅读顺序

1. 需要下载整合包：先阅读 [AI 整合包下载器](../tools/portable-downloader.md)。
2. 已经下载整合包，想本地免安装使用 WebUI：阅读 [整合包下载与使用](./portable.md)。
3. 希望用图形界面接管 Installer 生成的整合包管理脚本：阅读 [Windows GUI Launcher](../tools/launcher-gui.md)。
4. 成功启动 WebUI 后，需要学习界面使用、模型使用、工作流和绘世启动器操作：阅读 [SD Note](https://licyk.github.io/SDNote/)。
5. 使用 AMD 显卡并需要在 Windows 上运行：阅读 [AMD ZLUDA 使用](./zluda.md)。
6. 启动脚本或绘世启动器遇到问题：阅读 [常见问题](./qa.md)。

## 文档说明

- [整合包下载与使用](./portable.md)：包含各产品整合包入口和常用管理脚本。
- [AI 整合包下载器](../tools/portable-downloader.md)：介绍下载器界面、下载源、队列、自动解压和常见操作。
- [AMD ZLUDA 使用](./zluda.md)：包含 HIP SDK、PyTorch 重装、780M / 680M 额外配置、绘世启动器启动和跑图测试。
- [常见问题](./qa.md)：记录本地运行时常见问题和处理方式。
