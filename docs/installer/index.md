# 安装器使用

SD WebUI All In One 的安装器用于快速安装和管理常见的 AI WebUI 与训练工具。用户只需要完成基础环境准备并运行对应的 PowerShell 安装脚本，就可以把 SD WebUI、ComfyUI、Fooocus、InvokeAI、Qwen TTS WebUI、SD Trainer 等工具部署到本地。

安装完成后，安装器会在目标目录中生成一组管理脚本，用于启动 WebUI、更新程序、打开已配置好的终端环境、下载模型、切换分支、重装组件、维护版本和排查环境问题。也就是说，安装器不只是一次性的安装脚本，而是后续启动和维护这些工具的本地管理入口。

这里按产品整理安装器文档。每个产品文档都按照新手安装路径组织：环境准备与安装、启动使用、模型或资源、维护更新、故障排查和常用命令。

安装器文档会帮助你把工具装好、启动起来，并理解生成的管理脚本。进入 WebUI 后，如果需要学习实际绘图、工作流、模型使用、提示词、绘世启动器操作或更细的使用问题，可以继续阅读 [SD Note](https://licyk.github.io/SDNote/)。
