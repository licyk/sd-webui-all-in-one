SD WebUI All In One☍

SD WebUI All In One 是一组用于安装、下载、启动和管理 AI WebUI / 训练工具的文档与脚本集合。你可以用它快速运行 Stable Diffusion WebUI、ComfyUI、Fooocus、InvokeAI、Qwen TTS WebUI、SD Trainer 和 sd-scripts 相关工具。

开始选择使用方式 云端运行 下载整合包 本地安装 Launcher 管理

选择你的使用方式☍

使用方式 入口 适合场景
云端运行 云端运行 不想配置本地环境，直接用 Colab / Kaggle 运行。
本地整合包 整合包快速开始 Windows 下载后解压使用，适合少配置快速启动。
Launcher 管理 Launcher 快速开始 用图形界面或终端统一安装、启动、更新和维护。
本地安装 本地安装 需要长期部署、完整环境配置和细粒度维护。
启动之后 启动后的下一步 下载模型、运行管理脚本、继续学习 WebUI 使用。

文档板块☍

  • 快速开始

    帮你在云端、本地整合包、本地安装和 Launcher 管理之间做选择，并给出最短启动路径。

    进入快速开始

  • 安装器使用

    使用 Installer 安装并管理 SD WebUI、ComfyUI、Fooocus、InvokeAI、Qwen TTS WebUI、SD Trainer 等工具。

    查看安装器文档

  • 下载器与启动器

    了解 AI 整合包下载器、Windows GUI Launcher、Bash TUI / CLI Launcher，统一处理下载、安装、启动和维护。

    查看工具文档

  • Jupyter Notebook

    在 Colab / Kaggle 中运行 WebUI 或训练工具，支持图形化参数配置，默认参数可直接运行。

    查看 Notebook 文档

  • 整合包与实用指南

    获取 Windows 免安装整合包，了解 AMD ZLUDA 使用方式，以及本地运行常见问题。

    查看整合包文档

  • 命令行工具

    使用 CLI 管理 WebUI / 训练工具，查看各产品命令、管理器命令和环境变量配置。

    查看命令行文档

  • 开发维护

    面向贡献者和后续维护者，记录 Python 内核、Patcher、Installer、Notebook 的架构、编码风格和验证方式。

    查看维护文档

启动之后☍

本项目文档主要覆盖安装、下载、启动和管理。WebUI 打开后，如果需要继续学习界面使用、绘图流程、模型使用、提示词、工作流或绘世启动器操作，可以阅读 SD Note。
